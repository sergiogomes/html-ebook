# História do HTML

## Objetivo da aula

Nesta aula, você vai entender:

- **Por que HTML virou a base da Web** (e por que continua sendo até hoje)
- **Como a simplicidade e compatibilidade** ajudaram o HTML a “vencer”
- **O que foi XHTML** e por que ele não se tornou o caminho principal da Web
- Como usar as **DevTools** para aprender HTML a partir de qualquer site

---

## 1) HTML é a base da Web

HTML é a base da maravilha chamada Web.

Existe um poder incrível por trás desse conjunto de regras relativamente simples (e até “limitado”), porque é isso que permite criar documentos, aplicativos e experiências para pessoas no mundo inteiro.

Apesar de ter sido lançado em 1993, mais de 30 anos se passaram e o HTML ainda é a base da Web - com mudanças bem pequenas quando você compara com outras tecnologias.

Claro: hoje temos mais tags semânticas, e “HTML de apresentação” praticamente deixou de ser o padrão, porque **CSS** passou a cuidar do visual (CSS é o próximo módulo).

Isso reforça um ponto: o HTML é um sucesso por causa da **simplicidade**.

---

## 2) O desvio do caminho: XML e o tal do XHTML

Em algum momento, tentou-se incorporar o HTML ao mundo do XML (um outro tipo de linguagem de marcação que aparece muito em sistemas e documentos estruturados - por exemplo, notas fiscais eletrônicas).

Nessa ideia, o HTML “viraria” **XHTML** (*eXtensible HyperText Markup Language*).

O problema é que, na prática, isso adicionava complexidade demais para a Web do mundo real.  
Eventualmente, a comunidade percebeu que o caminho ficava pesado e burocrático - e o HTML seguiu evoluindo sem precisar “virar XML”.

Depois que você aprende algumas regras, você ganha muita liberdade.

---

## 3) A decisão mais importante da Web: compatibilidade

A plataforma web fez uma coisa muito certa: ela sempre manteve compatibilidade com versões anteriores.

Isso é quase um “superpoder”: dá pra voltar e olhar documentos HTML escritos em 1991 e entender boa parte do que está acontecendo.  
Essa estabilidade é um dos motivos de a Web ter crescido tanto.

Até sabemos qual foi a primeira página da Web. É essa aqui:

<http://info.cern.ch/hypertext/WWW/TheProject.html>

Você pode colocar esse link na descrição da aula pra pessoa explorar.

E o mais legal: dá pra ver o **código-fonte** da página.

Esse é outro recurso fantástico da Web e do HTML: **podemos inspecionar o HTML de qualquer página**.  
Eu não conheço nenhuma outra plataforma que entregue essa capacidade de forma tão aberta.

---

## 5) DevTools: sua lupa pra aprender HTML

O nome disso, no dia a dia, é **DevTools** (*ferramentas de desenvolvedor*).

Elas vêm embutidas em qualquer navegador moderno e permitem:

- Inspecionar a estrutura HTML de qualquer site
- Ver classes, ids e atributos
- Entender como um layout foi construído
- Aprender observando padrões reais

Se você está começando, isso acelera muito seu aprendizado: você não aprende só “no papel”, aprende no mundo real.

---

## 7) Fechamento

- HTML é simples - e essa simplicidade ajudou a Web a escalar.
- A Web priorizou compatibilidade, e isso foi decisivo.
- XHTML foi uma tentativa “mais rígida”, mas complexa demais para a Web real.
- DevTools é a sua ferramenta número 1 para aprender olhando sites reais.
